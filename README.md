# TripleTap Pro — Website

Marketing and legal pages for [tripletapapp.com](https://tripletapapp.com).

## Pages

- `/` — Home / landing page
- `/support` — Support & FAQ
- `/privacy` — Privacy Policy

## Deployment

This site is hosted via GitHub Pages with a custom domain.

DNS records point `tripletapapp.com` to GitHub Pages servers.
